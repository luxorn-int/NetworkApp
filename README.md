# NetworkApp

A mini project to create a Client -> Server application. 
Allows you to send text messages to the server, as well as transfer files.

## Prepare
- Set your **Boost** location in cmake module `cmake/includeBoost.cmake`

## Desciption:

### TECH STACK:
- **C++ 17**
- **Boost.Asio**
- **CMake**
- **GTest**
- **GMock**
